# Dual-Torabo-Tsuki-LP改（仮）

torabo-tsuki lpは、せきごん氏によって設計された、トラックボール搭載の薄型（ロープロファイル）分割型ワイヤレスキーボードです。
本プロジェクトでは、この torabo-tsuki lp をより自分好みに最適化するため、PCB（プリント基板）の設計変更やケースの改造に取り組むことを目的としています。

## やりたい事
- 自分はMサイズを購入して使用しているが、Sサイズに近い構成で制作する。
- 電池のプラスマイナスを逆に取り付けられる様に配線を変更する。（縦型デザインの乾電池が正しい向きで見える様にする）
- トラックボールはデュアル前提とする。
- その上で、親指キーを左右３つずつ配置する
- トラックボールは19mm球を前提とする。
- ボトムプレート、トラボケースをケースと一体にして、高さ的にトラボが出っ張らない様にする。
- 電池カバーをもっと開閉しやすくする。（現状ノーアイデア）
- 持ち運び用のカバーを左右入れ替えて底側から被せるとテンティングになる。みたいな感じのやつ
